# How this work
## Prerequis
[Python 3.8 or higher](https://www.python.org/) and [Git](https://git-scm.com/download/)

## Installation

1. Clone the repo with git
```bash
git clone https://github.com/ytpzh/four-in-a-row_discord.py/
```

2. Go to the repo
```bash
cd four-in-a-row_discord.py
```
3. Install discord.py
```bash
pip install discord.py
```
## Start the bot

1. Create the `.env` file and enter your bot token in it using the format `TOKEN=token`
2. Run the bot
```bash
python index.py
```
